01

nmap คืออะไร

Nmap (Network Mapper) คือเครื่องมือ Open Source สำหรับวิเคราะห์เครือข่าย พัฒนาโดย Gordon Lyon (Fyodor) ใช้งานฟรีภายใต้ NPSL/GPL License

ความสามารถรายละเอียด
Host Discoveryสแกนหา Host ที่เปิดอยู่ในเครือข่าย
Port Scanningค้นหา Port ที่เปิดอยู่ (Open Ports)
OS & Version Detectionตรวจจับ OS และ Service Version
NSE Scriptsตรวจหาช่องโหว่ด้วย Lua Scripts
Network Auditทำ Network Inventory และ Audit
⚠️
ข้อควรระวังด้านกฎหมาย
ใช้สแกนเฉพาะระบบที่ได้รับอนุญาตเท่านั้น • การสแกนระบบผู้อื่นโดยไม่ได้รับอนุญาต = ผิด พ.ร.บ. คอมพิวเตอร์ • ฝึกใน Lab, VM, HackTheBox หรือ TryHackMe

03

รูปแบบคำสั่งพื้นฐาน

syntax
nmap [Scan Type] [Options] {target}

# {target} = IP เดี่ยว, IP range, subnet, hostname, หรือ file

ตัวอย่าง: สแกนเป้าหมายเดียว

output
$ nmap 192.168.1.1

Starting Nmap 7.94 at 2024-01-15 10:30
Nmap scan report for 192.168.1.1
Host is up (0.0023s latency).

PORT      STATE    SERVICE
22/tcp    open     ssh
53/tcp    open     domain
80/tcp    open     http
443/tcp   open     https
8080/tcp  open     http-proxy

Nmap done: 1 IP address (1 host up) scanned in 1.52 seconds

ความหมายของ STATE

Stateความหมาย
openเปิดอยู่ มี service รับ connection
closedปิดอยู่ แต่ host up
filteredfirewall บล็อก nmap ไม่รู้สถานะ
open|filteredไม่แน่ใจว่า open หรือ filtered
unfilteredport accessible แต่ไม่รู้สถานะ
