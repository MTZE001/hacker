01

SQLMap คืออะไร?

SQLMap เป็นเครื่องมือ Open Source สำหรับทดสอบช่องโหว่ SQL Injection อัตโนมัติ มีความสามารถครอบคลุมตั้งแต่การตรวจจับจนถึงการ exploit

  • ตรวจจับและ Exploit ช่องโหว่ SQL Injection
  • รองรับฐานข้อมูลหลายประเภท: MySQL, PostgreSQL, MSSQL, Oracle, SQLite และอื่นๆ
  • ดึงข้อมูลจากฐานข้อมูล: ชื่อ DB, ตาราง, คอลัมน์, ข้อมูล
  • Bypass WAF, Firewall, IDS/IPS ด้วย Tamper Scripts
  • เข้าถึง OS Shell, อ่าน/เขียนไฟล์บนเซิร์ฟเวอร์
02

ติดตั้ง SQLMap บน Kali Linux

SQLMap มาพร้อมกับ Kali Linux อยู่แล้ว แต่ถ้าต้องการติดตั้งหรืออัปเดต:

bash 
# ตรวจสอบว่ามีอยู่แล้วหรือไม่
$ sqlmap --version

# ติดตั้งผ่าน apt
$ sudo apt update && sudo apt install sqlmap -y

# หรืออัปเดตจาก GitHub (เวอร์ชันล่าสุด)
$ git clone https://github.com/sqlmapproject/sqlmap.git
$ cd sqlmap
$ python3 sqlmap.py --version
output

03

โครงสร้างคำสั่งพื้นฐาน

syntax 
sqlmap [OPTIONS] -u "URL" [PARAMETERS]
-u / --url
กำหนด URL เป้าหมาย
-p
กำหนด parameter ที่ต้องการทดสอบ
--data
ส่งข้อมูลแบบ POST
--cookie
ส่ง Cookie
--level
ระดับความเข้มข้น (1-5, ค่าเริ่มต้น=1)
--risk
ระดับความเสี่ยง (1-3, ค่าเริ่มต้น=1)
--batch
ตอบ Yes อัตโนมัติทุกคำถาม
--dbs
แสดงรายชื่อฐานข้อมูลทั้งหมด
--tables
แสดงตารางในฐานข้อมูล
--columns
แสดงคอลัมน์ในตาราง
--dump
ดึงข้อมูลทั้งหมดในตาราง
-D / -T / -C
กำหนดชื่อ Database / Table / Column
